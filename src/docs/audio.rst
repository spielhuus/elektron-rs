audio
=====

.. toctree::
   :maxdepth: 1
   :hidden:

The notebooks audio command can be used to write calculated audio data and include it in the notebook.

.. code-block:: md

    ```{audio data="py$sample1", samplerate="py$fs"}```

**Variables**

* **data**: The audio data.
* **ext**: The file extension [default=wav].
* **samplerate**: The samplerate.
